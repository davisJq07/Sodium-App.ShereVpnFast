# How to share your VPN connection .using our app: The app offers two sharing 

methods; you can use them simultaneously or choose just one for greater flexibility and ease of use. The first method uses HTTP: Enable the hotspot on Phone A and activate our app by tapping "Start." On Phone B, go to network settings and connect to that hotspot. Then, edit the network settings, select "Manual Proxy Configuration," and enter the IP address displayed in the app (or retrieve it from the network gateway). In the second field, enter the port number—8080—which is also shown in the app. 

The second method for enabling UDP support is via SOCKS5. This requires installing an app—such as NekoBox, socksDroid, or any app that supports SOCKS configuration—on Phone B. You then manually configure the SOCKS settings: enter the IP address (obtained from the network gateway or the app on Phone A) and the standard port (1080). Finally, simply tap "Connect" within the app—without altering any system network settings—and the setup is complete.
